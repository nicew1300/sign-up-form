# sign-up-form
This project was a responsive sign-up form made with HTML and CSS. The most challenging part was making two columns of input fields with proper spacing. I used flex-wrap and widths slightly under 50% to leave room for gaps between inputs. This project helped me get better at styling forms and handling responsive layouts with Flexbox.
In this project, the newly used technologies were CSS flex-wrap, responsive input layouts, and form styling.
