# sign-up-form
Description:
Responsive sign-up form project built with CSS and HTML. Focused on layout, input handling, and modern CSS units.

Key Features:

Two-column input layout using Flexbox.

Proper spacing and responsive input sizing.

Styled form elements with pseudo-classes and CSS transitions.

Challenges & Learnings:

Practiced responsive design with complex input layouts.

Learned to manage input widths and gaps for multi-column forms.
